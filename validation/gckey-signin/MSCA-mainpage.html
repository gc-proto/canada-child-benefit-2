Canada Pension Plan (CPP) / Old Age Security (OAS) Pension

Notice

When applying for CPP retirement pension, you may receive the following error message: "System Unavailable, we are currently experiencing technical difficulties. Please try again later, we apologize for any inconvenience". If you get this error, please verify the following:

  • the area code/country code is less than three digits. Three digits must be entered in the field.
  • a telephone number has been entered. The home phone field must be filled, a cell phone number can be used.
  • if you request a Federal Voluntary Tax Deduction, request a percentage amount with NO decimal point (I.e.: 25% not 25.2%).
